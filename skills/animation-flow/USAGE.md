# animation-flow 使用指南

> 面向与 AI 对话的使用者。快速了解怎么触发、要给什么、每步会发生什么。

## 这是什么

把预置的网页动画效果落地成**你项目里的组件**。你描述想要的效果，AI 自动探测你的项目框架（React / Vue / Svelte / 原生），把效果转成可直接用的组件代码。即使你没说"动画"两个字，只要意图匹配也会触发。

覆盖五类效果：数字滚动、Hover 光效、手风琴、定屏滚动、文字打字。

## 怎么触发

直接用自然语言描述你要的效果，命中以下关键词即可触发（中英文都行）：

| 效果 | 关键词（部分） |
|------|----------------|
| 数字滚动 | 数字滚动、counter、数字递增、统计动画、数据滚动 |
| Hover 光效 | hover 光效、扫光、卡片光效、鼠标跟随光效、产品卡片 |
| 手风琴 | 手风琴、折叠面板、悬停展开、横向展开、类别展示 |
| 定屏滚动 | 定屏滚动、sticky scroll、左固右滚、企业文化页面、关于我们页面 |
| 文字打字 | 打字机、打字效果、typing effect、逐字动画、终端打字 |

> 没匹配上也没关系——AI 会列出所有可选效果让你挑。

## 五类效果速查

| 效果 | 你说什么（示例） | 你要提供什么数据 | AI 会输出什么 | 要装依赖吗 |
|------|------------------|------------------|---------------|-----------|
| 数字滚动 | "数据展示区，数字从 0 滚到目标值" | 每项：数字、后缀（如 +/%）、说明文字 | NumberScroll 组件 + 使用示例 | 否 |
| Hover 光效 | "产品卡片 hover 要有光线扫过" | 每项：图片、标题、描述 | HoverGlow 组件 + 使用示例 | 否 |
| 手风琴 | "4 个类别横向排列，hover 展开" | 每项：图片、标题、描述 | Accordion 组件 + 使用示例 | 否 |
| 定屏滚动 | "关于我们页，左边固定右边滚动" | 左侧标题/副标题 + 右侧图片卡片列表 | FixedScroll 组件 + 使用示例 | 是，gsap |
| 文字打字 | "首页标语打字机效果，轮流展示几句话" | 多行文本 | Typewriter 组件 + 使用示例 | 否 |

组件统一遵循：Props 驱动（数据不硬编码）、完整 TS 类型、样式自包含、响应式 + `prefers-reduced-motion` 降级。

## 一次完整用法

以数字滚动为例：

1. **你说**：「我官网要一个数据展示区，展示 30+ 国家、5000+ 客户、99.9% 可用性、200+ 合作伙伴，数字从 0 滚到目标值。」
2. **AI 识别** 到数字滚动效果，读取对应模板。
3. **AI 探测框架**：检查 package.json 等判断你的项目是 React / Vue / Svelte / 原生；判断不出会直接问你。
4. **AI 输出**：完整组件代码（含 4 项数据的 Props 类型）+ 调用示例 + 安装命令（本例无依赖）+ 落地提示（文件放哪、样式方案）。
5. **AI 跟进**：问你需不需要调整颜色、动画时长或数据来适配项目风格。

> 其它效果流程一样：描述 → 探测框架 → 输出组件 → 跟进调整。

## 触发不到怎么办

如果你的描述没命中任何关键词，AI 会读取所有可选效果并**列出清单让你选**，不会瞎猜。换一种说法或直接说效果名（如"我要数字滚动"）也行。

## 你需要注意

- **框架探测不出时会问你**：同目录有多个框架特征时，AI 会明确询问目标框架，别忽略。
- **定屏滚动需要装 gsap**：AI 会按你项目的包管理器（npm/pnpm/yarn/bun）给出对应安装命令。
- **不适用场景**：动画/动效调试（Framer 卡顿、CSS bug）、路由切换过渡、loading spinner、数据图表、轮播图、SVG 描边、Three.js/3D 粒子、纯后端任务——这些不在本技能范围。
- **数据只出现在调用方**：组件本身不硬编码数据，示例数据放在你调用的地方，方便替换。

## 想加新效果

如果五个效果都不够用，直接说「加个 XX 效果」（如"加个跑马灯效果"），AI 会走三阶段确认流程：

1. **定义** — 和你确认效果定义卡（关键词、数据结构、可调参数），你确认后才实现。
2. **实现** — 产出可浏览器预览的 `effect.html` + `README.md`，你打开预览确认视觉效果。
3. **集成** — 更新技能索引让新效果能被关键词匹配，告诉你新增的关键词。

每阶段末都会等你确认，不会跳过。
