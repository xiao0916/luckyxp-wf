# 文字打字 (Typewriter)

- **关键词**: typewriter, 打字效果, 打字机, typing effect, text typing, 文字逐字显示, 逐字动画, 文字动画, 打字动画, 终端打字
- **分类**: 文字动画
- **描述**: 模拟打字机效果，文字逐字输入并循环展示多行文本，配光标闪烁。适合首页标语、代码展示、终端风格界面等场景。纯 CSS + 原生 JS 实现，无额外依赖。
- **依赖**: 无

## 数据结构

组件接收一个文本字符串数组：

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| text | string | 是 | 要逐字打字展示的文本内容 |

## 可调参数

| 参数 | 类型 | 默认 | 说明 |
|------|------|------|------|
| typeSpeed | number | 100 | 打字速度 (ms/字符) |
| deleteSpeed | number | 50 | 删除速度 (ms/字符) |
| pauseDuration | number | 2000 | 打完一行后停留时间 (ms) |
| className | string | "" | 容器自定义类名 |

## 转换提示

- 效果实现见 `effect.html`（原生 JS + CSS，浏览器可预览）
- 打字循环逻辑：逐字输入 → 停留 → 逐字删除 → 切换下一行，循环往复
- 光标闪烁通过 CSS `@keyframes` 实现，转换时随组件样式走
- 转换成目标框架组件时，遵循 `component-spec.md` 的规范约束
